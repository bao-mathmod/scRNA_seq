# Content license note

The MIT license in this repository is intended for code.

For teaching text, slides, diagrams, and figures, choose a separate content license before making the repository public.

Suggested options:

- CC BY 4.0 if you want broad reuse with attribution.
- CC BY-NC 4.0 if you want non-commercial reuse with attribution.
- All rights reserved if the material includes third-party slides or figures with unclear redistribution rights.

Do not publish third-party slide decks or figures unless you have permission or a compatible license.
